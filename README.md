# NutriShare

API REST para la aplicación NutriShare - Sistema de gestión nutricional.

## Tecnologías

- Java 21
- Spring Boot 3.5.7
- MongoDB
- Spring Security con JWT
- Maven
- Swagger/OpenAPI

## Requisitos

- Java 21
- Maven 3.6+
- MongoDB instalado
- Navegador web

## Instalación

```bash
# Compilar y ejecutar
mvn clean install
mvn spring-boot:run
```

## Configuración

La aplicación utiliza MongoDB como base de datos y corre en un servidor local.

## Testing

```bash
mvn test
```

## Build

```bash
mvn clean package
```
